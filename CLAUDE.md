## **💰 СХЕМА РАСЧЁТА ЦЕН**

### **Фиксированные расходы на товар:**
- Закупочная цена: 1,000₽ (переменная)
- Упаковка + доставка от поставщика: 200₽
- Доставка до клиента: 400₽

### **Переменные расходы (% от цены):**
- Комиссия платформы: 12%
- Страховка: 5%
- Налог УСН (доход-расход): 20%

### **Целевые показатели:**
- Минимальная маржа: 30%
- Идеальная маржа: 35-40%

### **Пример:** Товар закупка 1,000₽ → Продажа 2,500₽ (маржа 30%)

## **🤖 AI АГЕНТЫ**

### **1. Parser Agent** 🛒
**Задача:** Синхронизация товаров с ctradei
- Автоматический вход на ctradei
- Скачивание обоих файлов
- Merge файлов по ID товара
- Генерация unified YML/CSV

### **2. Pricing Agent** 💰
**Задача:** Автоматический расчёт и управление ценами
- Расчёт оптимальной цены (30% маржа)
- Применение правок к ценам
- Анализ конкурентов (при необходимости)

### **3. SEO Agent** 🔍
**Задача:** Оптимизация контента товаров
- Анализ ключевых слов
- Оптимизация описаний
- Улучшение видимости

### **4. Supplier Agent** 🏭
**Задача:** Управление поставщиками
- Мониторинг цен поставщика
- Поиск альтернативных поставщиков

### **5. Product Agent** 📦
**Задача:** Анализ и выбор товаров
- Анализ спроса
- Прогнозирование продаж

## **⚙️ API ЭНДПОИНТЫ**

### **Управление ценами:**
```bash
POST /api/pricing/calculate
{
  "cost": 1000,
  "targetMargin": 0.30
}
```

### **Добавить товар:**
```bash
POST /api/products/add
{
  "id": "SKU123",
  "name": "Кровать King",
  "description": "...",
  "cost": 1000,
  "quantity": 50
}
```

### **Изменить цену:**
```bash
PUT /api/products/SKU123/price
{"newPrice": 2500}
```

### **Обновить остатки:**
```bash
PUT /api/products/SKU123/quantity
{"quantity": 35}
```

## **📅 РАСПИСАНИЕ АВТОМАТИЗАЦИИ**

- **Parser Agent:** Каждый день в 00:30 MSK
  - Скачивает CSV (остатки) с https://ctradei.com/f/ostatki_2020.csv
  - Скачивает YML (описания) с https://ctradei.com/x/shop2_1410641-yml.xml
  - Merge файлов по product_id
  - Обновление товаров в feedsGenerator
  
- **Pricing Agent:** Автоматически при обновлении товаров
  - Расчёт Smart Price для каждого товара
  - Применение 30% маржи
  - Сохранение в YML фид

- **Sync to insales:** Автоматически
  - insales берёт товары с YML ссылки: `/api/feeds/yml`
  - Обновляет каталог товаров
  - Синхронизирует цены
