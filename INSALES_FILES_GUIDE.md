# 🛒 Импорт файлов в insales - Полное руководство

Система автоматической генерации CSV и YML файлов для загрузки товаров в insales.

## 📋 Почему файлы вместо API?

| Параметр | Файлы | API |
|----------|-------|-----|
| **Простота** | ✅ Просто загрузить файл | ❌ Нужны ключи |
| **Скорость** | ✅ Быстро | ✅ Быстро |
| **Ошибки** | ✅ Видны все | ⚠️ Логи сложнее |
| **Контроль** | ✅ Полный контроль | ⚠️ Автоматизм |
| **Изображения** | ✅ Все сразу | ⚠️ По одному |

**Вывод:** Файлы - проще, надежнее, понятнее для пользователя! 📁

### Шаг 1: Получить файлы (автоматически каждый день)

Файлы генерируются автоматически в 00:30 MSK и доступны по ссылкам:

```

📋 YML файл:  https://grachik2007.github.io/TestGitHub/insales-import.yml
📄 Инструкция: https://grachik2007.github.io/TestGitHub/insales-import.html
```


```
Личный кабинет insales → Товары → Импорт товаров
↓
```

- ✅ **Полная информация** - все параметры
- ✅ **Структурированность** - понятный XML
- ✅ **Совместимость** - работает с insales

## 🔄 Расчет цен (Smart Pricing)

Каждая цена в файлах рассчитана по одной формуле:

```
Розничная цена = (Цена поставщика + Расходы) × (1 + Маржа)
```

### Подробный расчет

```
Товар: Комплект постельного белья
├─ Цена поставщика (ctradei):   2999₽
├─ Логистика (8% от цены):      + 240₽
├─ Комиссия (5% от цены):       + 150₽
├─ Фиксированные расходы:       + 15₽
├─ = Себестоимость:             = 3404₽
│
└─ Маржа (35% от себестоимости): × 1.35
   └─ = РОЗНИЧНАЯ ЦЕНА:         = 4595₽
```

### Компоненты расчета (настраиваются)

| Компонент | Значение | Что это |
|-----------|----------|--------|
| Маржа | 35% | Ваша прибыль |
| Логистика | 8% | Доставка поставщика |
| Комиссия | 5% | Комиссия маркетплейса |
| Фиксированные | 15₽ | Прочие расходы |

**Где менять:** `.github/workflows/generate-feed.yml`

```yaml
PRICING_MARGIN: '35.0'           # Маржа
PRICING_LOGISTICS: '8.0'         # Логистика
PRICING_COMMISSION: '5.0'        # Комиссия
PRICING_FIXED: '15.0'            # Фиксированные
```

## 📥 Пошаговая инструкция импорта

### Вариант 1: Импорт CSV (быстрый способ)

```
1. Скачайте файл insales-import.csv
   └─ https://grachik2007.github.io/TestGitHub/insales-import.csv

2. Откройте личный кабинет insales
   └─ https://www.insales.ru/

3. Перейдите в Товары
   └─ Левое меню → Товары

4. Нажмите "Импорт товаров"
   └─ Кнопка вверху или в меню

5. Выберите тип файла: CSV

6. Загрузите файл insales-import.csv
   └─ Кликните на поле "Выбрать файл"

7. Проверьте соответствие колонок
   └─ Система должна автоматически определить
   └─ Название → Название
   └─ Артикул → Артикул/SKU
   └─ Цена → Цена
   └─ и т.д.

8. Нажмите "Импортировать"

9. Дождитесь завершения
   └─ Можно следить за прогрессом
```

### Вариант 2: Импорт YML (с изображениями)

```
1. Скачайте файл insales-import.yml
   └─ https://grachik2007.github.io/TestGitHub/insales-import.yml

2-6. Все как в Варианте 1, но выберите YML

7. Нажмите "Импортировать"
   └─ Insales загрузит все изображения

8. Дождитесь завершения
   └─ С изображениями дольше (10-30 мин)
```

## 🔄 Автоматическое обновление

### Как это работает

```
GitHub Actions
    ↓
Каждый день в 00:30 MSK
    ↓
1. Получить YML с ctradei (изображения, описания)
2. Получить CSV с ctradei (цены, остатки)
3. Объединить по ID товаров
4. Применить Smart Pricing расчеты
5. Сгенерировать insales-import.csv
6. Сгенерировать insales-import.yml
7. Публиковать на GitHub Pages
8. Вы скачиваете и загружаете в insales
```

### Периодичность

| Источник | Обновление | Примечание |
|----------|-----------|-----------|
| ctradei YML | Каждые 15 мин | На ctradei |
| ctradei CSV | В реальном времени | Живые остатки |
| Наши файлы | Раз в день (00:30 MSK) | Регулярно |

## 🚨 Решение проблем

### Проблема: "Ошибка при загрузке файла"

```
Возможные причины:
1. Файл поврежден → Скачайте заново
2. Неверный формат → Убедитесь что CSV/YML
3. Кодировка → Файл должен быть UTF-8
4. Слишком большой → Попробуйте меньший файл

Решение:
- Скачайте файл заново
- Используйте CSV вместо YML
- Проверьте кодировку в Excel: UTF-8 с BOM
```

### Проблема: "Товары не импортировались"

```
Возможные причины:
1. Неверное соответствие колонок
2. Неверные данные в файле
3. Товары уже существуют

Решение:
- Внимательно проверьте соответствие колонок
- Скачайте файл заново
- Если товары дублируются → они обновятся по артикулу
```

### Проблема: "Цены неправильные"

```
Возможная причина:
- Валюта установлена на иную

Решение:
1. Перейдите в Настройки → Валюта
2. Установите RUB (российский рубль)
3. Переимпортируйте файл
```

### Проблема: "Изображения не загружаются"

```
Возможные причины:
1. Используется CSV (изображения только по 1)
2. Интернет медленный (YML загружается долго)
3. Изображение удалено на ctradei

Решение:
- Используйте YML для полноты изображений
- Дождитесь завершения импорта (может быть 10-30 мин)
- Проверьте что изображения видны в YML файле
```

## 📈 Когда переимпортировать

Переимпортируйте файл когда:

- ✅ **Каждый день** - для актуального остатка
- ✅ **При изменении цен** - маржу изменили
- ✅ **При добавлении товаров** - новые товары с ctradei
- ✅ **При получении изображений** - если были пусты

**Как это работает:**
- Товары будут искаться по артикулу (ID)
- Если найден → обновится
- Если новый → создается
